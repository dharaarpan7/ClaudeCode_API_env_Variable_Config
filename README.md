## 📌 About

The `.claude` directory stores configuration files required for setting up the Claude Code environment variables.

The `settings.json` file is used to define environment variables that allow your code to connect to an API provider like OpenRouter.

---

├── .claude/
 └── settings.json

## ⚙️ Setup

To configure your environment, you only need two things:

- OpenRouter API Key  
- Model Name  

---

## 🛠️ Configuration Steps

1. Open the `.claude/settings.json` file  
2. Edit the configuration
